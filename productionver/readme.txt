This is just a folder to put a version of index and a merged, minified version of the source text.


*****************
To minify:
open polyomino.min.js
delete everything in the file
open index.html from the NON production version folder
copy contents of all JS files into polyomino.min.js, in the order specified in the index.html
copy contents of polyomino.min.js to this website:
	http://jscompress.com/
copy minified code back into polyomino.min.js and save
******************


host resources folder, index.html, and polyomino.min.js on a website

Enjoy a 10x faster website!
